# DualWallpaper System Module

## Overview

This module installs **DualWallpaper** as a **system application** using **KernelSU, Magisk, APatch, or compatible root solutions**.

Features:

- System app installation
- Privileged permissions support
- Wallpaper integration
- Background persistence helper
- Battery optimization protection
- Android 8+ support
- HyperOS compatible

This module:

- Does NOT modify the original application code
- Does NOT claim ownership of the original application
- Provides system-level integration only

## Optional Requirement (Mountify)

Repository:

https://github.com/backslashxx/mountify

Install Mountify only if:

- App does not appear after reboot
- System mounting fails
- ROM blocks overlays

Mountify is NOT bundled.

## Installation

1. Install root solution
2. Flash module ZIP
3. Reboot device

Install Mountify if required.
